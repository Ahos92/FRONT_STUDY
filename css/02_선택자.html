# CSS 선택자(Selector)를 통해 원하는 요소를 정확하게 선택할 수 있어야 한다

  • tag : 해당 HTML태그를 모두 선택

    • # ul의 형제는 검은색으로 #

  • #id : 해당 요소의 ID속성을 이용해 선택
  • .className : 해당요소의 클래스 속성을 이용해 선택
  • ※ 클래스는 여러개 적용할 수도 있다.
  • + : 형제 요소를 선택한다
    ex) div + ul : div바로 다음 형제 ul만 선택한다. (div와 형제가 아닌 ul은 선택되지 않음)
  • 공백 : 해당 요소의 모든 자손 요소들을 선택한다.
  • > : 해당요소의 직계 자손 요소들만 선택한다.
  • * : 모든 요소를 선택한다
